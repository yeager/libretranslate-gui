# LibreTranslate GUI

Translation assistant using the LibreTranslate API.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install libretranslate-gui
```

### Fedora/RPM
```bash
sudo dnf install libretranslate-gui
```
